# Terra_Project: PDSI Drought Analysis

Drought analysis for West Des Moines, Iowa (2022-2025) using the Palmer Drought Severity Index (PDSI).

### Contents:
* **Data:** 5-day pentads from [Climate Engine](https://app.climateengine.org/climateEngine) (gridMET).
* **Processing:** Temporal aggregation from high-resolution data to monthly averages.
* **Notebook:** `PDSI_Analysis_Iowa_2022_2025.ipynb` containing cleaning and trend visualization.

### Tools:
* Python (Pandas, Matplotlib)
* Jupyter Notebook
