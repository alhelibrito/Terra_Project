## Start Local Web Server
Follow the instructions in [config.example.js](config.example.js).

From within this directory, run `make serve` in the terminal.  
This will build `data.js` from the CSV and start the server.  
You will then be able to see the web app at http://localhost:8765 in your browser.

## Stop Local Web Server
In the same terminal, hit `Ctrl+C`
